### FLASK API WITH DATABASE

#### Librerias a utilizar:

- flask
- flask-migrate
- flask-sqlalchemy
- flask-cors
- python-dotenv
- psycopg2-binary (PostgreSQL)
- pymysql (MySQL)
- sqlite


#### Exportar comando flask al terminal

Windows:
```shell
SET FLASK_APP=src/app.py
```

Linux or Mac:
```shell
export FLASK_APP=src/app.py
```
